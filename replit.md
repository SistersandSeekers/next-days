# Next Days & Priorities

## Overview
A static web application for Sisters and Seekers that helps check replenishment data. Users can paste "next days" text and upload CSV files to check replenishment information.

## Project Structure
```
├── index.html          # Main HTML page
├── assets/
│   ├── css/
│   │   └── style.css   # Stylesheet
│   ├── images/
│   │   ├── S_S_Logos-01_320x.avif  # Logo image
│   │   └── truck-fast-solid.svg    # Favicon
│   └── js/
│       └── index.js    # JavaScript functionality
```

## Technology Stack
- Pure HTML/CSS/JavaScript (no build system)
- Font Awesome for icons (loaded via CDN)

## Development
The project is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment serving from the root directory.
